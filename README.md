# Blinkit Sales Dashboard Analysis

## Project Overview
This project analyzes Blinkit sales data using Power BI, Tableau, and Python. The dashboards provide business insights through interactive visualizations.

## Tools Used
- Power BI
- Tableau
- Python
- Jupyter Notebook
- Microsoft Excel

## Dataset
- BlinkIT Grocery Data.xlsx

## Dashboard Features
- Total Sales
- Total Profit
- Sales by Outlet
- Sales by Item Type
- Outlet Size Analysis
- Outlet Location Analysis
- Interactive Filters
